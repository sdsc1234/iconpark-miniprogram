# IconPark Mini Program Taro

---

> use in taro 3.x

## Installing

Using npm

`$ npm install @iconpark-miniprogram/taro`

Using yarn

`$ yarn add @iconpark-miniprogram/taro`

## Example

```
import {User} from '@iconpark-miniprogram/taro'

export default function Index() {
    return (<View>
        <User fill='#000' color='#fff' />
    </View>)
}

```
